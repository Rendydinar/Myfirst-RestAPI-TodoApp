# Myfirst-RestAPI-TodoApp
My First REST API Using NodeJs Todo-app. This REST-API use CRUD operations.

# Requirements
- NodeJs Installed
- MongoDB Installed
- Install Depencies

# Usege (Routes)
-    GET: /todo/:id
        - Menampilkan Todo Berdasarkan id 
-   POST: /todo
        - Menambahkan Todo
-    PUT: /todo/:id
        - Mengubah Todo berdasarkan id
- DELETE: /todo/:id
        - Menghapus Todo berdasarkan id
